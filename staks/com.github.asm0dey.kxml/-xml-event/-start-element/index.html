StartElement

data class StartElement(val name: String, val attributes: Map<String, String>, val attributeNamespaces: Map<String, String> = emptyMap(), val namespaces: Map<String, String> = emptyMap(), val prefix: String? = null, val namespaceURI: String? = null) : XmlEvent, XmlEvent.HasNameAndPrefix(source)

Represents the start of an XML element (opening tag).

This event is emitted when an opening tag is encountered in the XML document. It contains information about the element name, attributes, and namespace declarations.

Example:

<book id="1" xmlns:ns="http://example.com">
Content copied to clipboard

Would be represented as:

StartElement(
    name = "book",
    attributes = mapOf("id" to "1"),
    namespaces = mapOf("ns" to "http://example.com"),
    prefix = null,
    namespaceURI = null
)

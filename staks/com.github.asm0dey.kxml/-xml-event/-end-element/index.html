EndElement

data class EndElement(val name: String, val prefix: String? = null, val namespaceURI: String? = null) : XmlEvent, XmlEvent.HasNameAndPrefix(source)

Represents the end of an XML element (closing tag).

This event is emitted when a closing tag is encountered in the XML document. It contains information about the element name and namespace.

Example:

</book>

Would be represented as:

    name = "book",

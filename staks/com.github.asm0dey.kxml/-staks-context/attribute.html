suspend fun attribute(tagName: String, attributeName: String, tagNamespaceURI: String? = null, attributeNamespaceURI: String? = null): AttributeResult(source)

Gets the value of an attribute from a specific element.

This method extracts an attribute value from a specific element and returns an AttributeResult that provides type conversion methods. It's commonly used to get metadata or identifiers from XML elements.

Example:

// Get a string attribute value
val id = attribute("book", "id").string()

// Get an integer attribute value
val count = attribute("library", "count").int()

// Get a double attribute value
val price = attribute("product", "price").double()

// Get a boolean attribute value
val available = attribute("product", "available").boolean()

// Use the unary plus operator as a shorthand for .value()
val category = +attribute("book", "category")

val optionalAttr = attribute("element", "optional-attr").nullable().string()

Return

Parameters

tagName

The name of the tag that contains the attribute. Can include a namespace prefix (e.g., "ns:element").

attributeName

The name of the attribute to get the value from. Can include a namespace prefix (e.g., "ns:attr").

tagNamespaceURI

The namespace URI of the tag, or null to match any namespace

attributeNamespaceURI

The namespace URI of the attribute, or null to match any namespace
